# Support

Telemachus is a volunteer open-source project and does not provide guaranteed
support or response times.

For a reproducible defect, use the repository's bug-report template. For a
feature proposal, use the feature-request template. Security and privacy issues
must follow [SECURITY.md](SECURITY.md), not a public issue.

Before reporting a connection or performance problem:

1. Confirm the Mac and Android versions meet the requirements in the README.
2. Confirm `adb devices` lists the tablet as `device`, not `unauthorized`.
3. Try USB mode before wireless mode.
4. Record the selected resolution, refresh rate, codec, bitrate, reported FPS,
   dropped-frame count, and RTT.
5. Include relevant logs after removing device names, network addresses, and
   pairing data.

Questions about the original SideScreen project should be directed to that
project. Telemachus support channels should not be used to contact or solicit
the original author.
